# Vitamin D3 Glossary

## Terms

- 25(OH)D
  25-hydroxyvitamin D, the main blood marker used to assess vitamin D status.
- 1,25(OH)2D
  1,25-dihydroxyvitamin D, the active hormonal form. It is not the usual test for ordinary vitamin D status.
- Cholecalciferol
  Vitamin D3.
- Ergocalciferol
  Vitamin D2.
- Hypercalcemia
  High blood calcium. This is a central danger in vitamin D toxicity.
- Hypercalciuria
  High urine calcium. It can matter for kidney-stone risk.
- IU
  International Unit. For vitamin D, 40 IU equals 1 mcg.
- mcg
  Microgram. For vitamin D, 1 mcg equals 40 IU.
- Osteomalacia
  Adult bone-softening disease caused by defective mineralization, often linked to severe vitamin D deficiency.
- RDA
  Recommended Dietary Allowance: average daily intake sufficient for nearly all healthy people in a life-stage group.
- Rickets
  Childhood disease of defective bone mineralization, classically linked to vitamin D deficiency.
- UL
  Tolerable Upper Intake Level: highest average daily intake unlikely to pose risk for almost all people in a life-stage group. It is not a target dose.
- UVB
  Ultraviolet B radiation, the sunlight wavelength range that allows skin to produce vitamin D3.
