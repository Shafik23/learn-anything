# progress.md Format

`progress.md` is the compact learning-state file for a topic. It replaces a folder of learning records until the topic is large enough to need more structure.

## Template

```md
# {Topic} Progress

## Current Learning State

{1-3 sentences describing what the learner is trying to build, what can be assumed, and what should not yet be assumed.}

## Completed

- [Lesson 1: {Title}](./lessons/0001-title.md)

## Next

- {The next best concept, skill, or lesson}
```

## Rules

- Record durable learning state, not a session transcript.
- Update it when the learner demonstrates understanding, discloses important prior knowledge, corrects a misconception, or changes the mission.
- Keep private scratch notes out of this public file.
