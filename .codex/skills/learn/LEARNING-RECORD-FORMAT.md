# Learning Record Format

Learning records live in a topic directory under `./learning-records/` and use sequential numbering:

```text
0001-short-dash-case-title.md
0002-short-dash-case-title.md
```

They capture durable learning state. They are not session logs.

## Template

```md
# {Short title of what was learned or established}

{1-3 sentences: what was learned or what prior knowledge was established, and why it matters for future sessions.}
```

## Optional Sections

Use only when they add value:

- `Status: active | superseded by LR-NNNN`
- `Evidence`
- `Implications`

## When To Write One

Write a learning record when:

- the user demonstrated real understanding of something non-trivial
- the user disclosed prior knowledge
- a misconception was corrected
- the mission shifted

Do not write one for material that was merely covered.
